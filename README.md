## 🚀 Getting Started

This site is powered by simple HTML/CSS — no build steps required.

To run locally:

```bash
# Clone the repo
git clone https://github.com/petswithhannah/petswithhannah.github.io.git

# Open index.html in your browser
start index.html  # (or double-click it)

.\gitpush.bat
```
